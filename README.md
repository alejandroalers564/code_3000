Personal Statement: Ethics

I think that ethics should be focused on the impact of our choices. To be ethical we shouldn't just list values but define the specific things we refuse to let happen. Instead of hiding behind abstract words like "fairness," we need to identify our "ethical nightmares" and treat them as hard red lines. By starting with what is impermissible, we can build practical plans that actually prevent harm.
